HOW IT WORKS

1. You commit code
   ↓
2. Git hook fires (.git/hooks/post-commit)
   • Analyzes what changed
   • Stores diff for Claude
   • Prints suggestions
   ↓
3. Claude Code hook fires (hooks/post-commit-test-generation.json)
   • Detects git commit
   • Displays message
   • Suggests test generation
   ↓
4. You ask Claude Code: "Generate tests for my changes"
   ↓
5. Tests are created (automation/tests/)
   • Follows your patterns
   • Ready to run
   ↓
6. Run with: npm test

================================================================================

ADDING NEW THINGS

To Add a Hook:
  1. Create: .claude/hooks/my-new-hook.json
  2. Define the configuration
  3. Reference in settings.json (if needed)
  4. Restart Claude Code

To Add a Skill:
  1. Create: .claude/skills/my-skill.md
  2. Write documentation
  3. Restart Claude Code
  4. Use it: /my-skill
