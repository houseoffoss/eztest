# Code Reviewer Skill

## Description
Reviews code quality, identifies improvements, and simplifies complex logic.

## Trigger
Use manually on specific files or ask Claude Code to review code.

## What It Does
1. ✅ Analyzes code for quality issues
2. ✅ Identifies reusable patterns
3. ✅ Simplifies complex logic
4. ✅ Suggests performance improvements
5. ✅ Checks for common mistakes

## How to Use

### Review Specific File
```
/simplify src/components/MyComponent.tsx
```

### Ask Claude to Review
```
Review this component for quality issues
```

### Auto-Review After Changes
```
Review the code I just wrote and suggest improvements
```

## What It Checks
- ✅ Code duplication
- ✅ Naming conventions
- ✅ Performance issues
- ✅ Error handling
- ✅ Type safety
- ✅ Best practices

## Output
- Issue descriptions
- Before/after code examples
- Specific recommendations
- Implementation guidance

## Tips
- 💡 Use after significant refactoring
- 💡 Review before submitting PRs
- 💡 Great for catching bugs early

---

**Want to improve code quality?** Ask: "Review my code for improvements"
